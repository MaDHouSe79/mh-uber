# MH Uber
- a rp uber city job for qbcore.

# Commands
- '/ubermenu' or F4

# QB Sgared Job
```lua
uber = { label = 'Uber', defaultDuty = false, offDutyPay = false, grades = {['0'] = { name = 'Driver', payment = 0 } } },
```

# Add in qb-cityhall
- in config.lua in `Config.AvailableJobs`
```lua
['uber'] = { ['label'] = 'Uber', ['isManaged'] = false },
```
